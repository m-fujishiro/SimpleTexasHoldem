【タイトル】
	TNW式ポーカー
【製作期間】
	
【使用言語】
	PHP,JavaScript,HTML/CSS,SQL(MySQL)
【ジャンル】
	カジノゲーム
【概要】
	ポーカー(テキサスホールデム)の賭け方を簡素にしたもの
【操作方法】
	ベット数を決める際はキーボード入力かマウス選択でベット数を選びエンターキーで決定
	任意の操作のボタンをマウスクリック
【ゲームオーバー】
	チップがなくなったらゲームオーバーです
【書いた範囲】
	全て
【制作動機】
	文化祭でやったものがとても楽しかったので再現してみました。
【頑張った所】
	シャッフルに乱数を一切使っていないところです。もちろんshuffle関数も使っていません。
【面白いところ】
	タダでカジノゲームが遊べるところです。あとはどこでリロードしても問題ないように組みました。
	本当のテキサスホールデムより賭け方が簡素になっているため、初心者でも遊びやすいです。
